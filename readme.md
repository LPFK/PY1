### 2. Event Planning App (`Planning_App.py`)
A class-based application for managing events, venues, and participants.
*   **Features:** Create events, assign halls (`Salle`), manage participants, and validate scheduling conflicts.
*   **Concepts applied:** Object-Oriented Programming (Classes `Evenement`, `Salle`, `PlanningManager`), Date/Time handling.
